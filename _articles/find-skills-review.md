## 测评概述

**find-skills** 是目前安装量最高的 AI Agent Skill，累计安装量超过 **160万次**。它由 Vercel Labs 开发，核心功能是帮助用户智能搜索和安装其他 Skills。

| 评测维度 | 评分 | 说明 |
|---------|------|------|
| 功能完整性 | ⭐⭐⭐⭐⭐ | 搜索、安装、推荐一体化 |
| 搜索准确性 | ⭐⭐⭐⭐ | 大部分场景准确，偶尔有偏差 |
| 安装便捷性 | ⭐⭐⭐⭐⭐ | 一条命令完成安装 |
| 使用体验 | ⭐⭐⭐⭐ | 响应速度快，交互简洁 |
| 文档质量 | ⭐⭐⭐⭐ | 有基本文档，缺少高级用法说明 |
| **综合评分** | **⭐⭐⭐⭐.5** | **强烈推荐** |

## 安装体验

### 安装命令

```bash
npx skills add vercel-labs/skills@find-skills
```

### 安装过程评估

- **耗时**：约 5-10 秒（取决于网络环境）
- **依赖**：无额外依赖，开箱即用
- **兼容性**：支持 Solo、Claude 等主流 Agent 平台

**评分：5/5** — 安装过程极其简洁，无需任何配置。

## 核心功能测试

### 1. 智能搜索

find-skills 的核心功能是语义搜索。我们测试了多种查询方式：

| 查询方式 | 示例 | 准确性 | 响应速度 |
|---------|------|--------|---------|
| 关键词搜索 | "frontend design" | 高 | <1s |
| 自然语言 | "帮我设计一个好看的网页" | 中高 | <2s |
| 功能描述 | "连接数据库的MCP" | 高 | <1s |
| 模糊搜索 | "写代码的工具" | 中 | <2s |

**亮点**：
- 支持中英文混合搜索
- 理解上下文语义，不只是关键词匹配
- 搜索结果按安装量和评分排序

**不足**：
- 对非常具体的细分需求，搜索结果可能不够精准
- 偶尔会推荐已过时的 Skill 版本

### 2. 一键安装

搜索到目标 Skill 后，find-skills 可以直接安装：

```
用户：帮我找一个代码审查工具
find-skills：推荐 grill-me（17.3万安装）
用户：安装它
find-skills：正在安装 mattpocock/skills@grill-me... 安装成功！
```

**评分：5/5** — 搜索到安装的流程无缝衔接。

### 3. 智能推荐

find-skills 还能根据用户的使用习惯推荐相关 Skill：

- 如果你经常使用前端相关 Skill，会推荐 UI/UX 设计类工具
- 如果你使用数据库 MCP，会推荐相关的数据分析工具

**评分：4/5** — 推荐逻辑合理，但推荐数量偏少（通常只推荐2-3个）。
